Рабочий функционал веб скады написанный на python, работающий в рамках одной wifif сети. 
Состоит из двух python приложений, расположенных на АРМ, общение этих микросервисов реализованно с помомщью kafka.
С помощью данных микросервисов данные с модбаса отправляются и приходят с вебсокет сервера, через который возможно подключение удаленных клиентов для мониторинга и управления ситемами.
Так же возможно локальное подключение по протоколу OPC UA не требующего подключения к интеренту.
