Данный микросервис опрашивает по протоколу сеть Modbus TCP, создает локальный сервер OPC UA и передает и считывает из него данные.
По протоколу mqtt с помощью kafka получает и передает данные в микросервис подключенный к вебсокет серверу.
Есть две версиии реализации данного микросервиса: в первом используется однопоточный опрос modbus устройств, во втором используется многопоточный опрос modbus устройств.
В данном репозитории представлен код многопоточного модбаса (При тестировании для каждого регистра был организован отдельный поток).
